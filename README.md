## Types of ML systems
1. **Supervised**
   1. **Regression**: predicts numeric value 
   3. **Calassification**: predicts likelyhood of something belonging to some category
3. **Unsupervised: Model**: is given data that does not contain correct answer instead it employs technique called as clustering which creates grouping of the data.
4. **Reinforcement learning**: It learns by getting reward or penalties based on the action performed
5. **Generative AI**: creates content from user input
